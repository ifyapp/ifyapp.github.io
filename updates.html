Welcome to Ify Updates! 

2023.8.26     v.0.1.6
What's New:
New Feature: Tasks Editing!
  • Want to edit your task for a better idea? No problem, Ify Tasks Editing has you covered! Just click the pen icon on the tool bar and start editing your text!
  • You can add or remove text, add emojis, and any symbol to your task item!
  • After finished editing the task, click anywhere else on the screen and the changes will be automatically saved!
New Feature: Tasks Drag & Drop!
  • In order to boost your productivity even more, you can now reorder tasks for your works and priority tasks!
  • Simply drag the task item to where you want it to be in the task list and drop it on the blue line indicating the position!
Improved User Interface and Functionalities
  • Fixed task managing interface and new task completing effect.
  • Appearance and effects improvements.
  • Improved sidebar navigation.
  • Bug fixes, including remove button animation, tasks dragging bug, tasks editing not working, and other functionality improvements.
  • Overall performance improvements and minor updates.

2023.8.2     v.0.1.3
What's New:
Introducing Ify: Let Your Ideas Fly!
  • Ify: Let Your Ideas Fly is an app that boosts your ideas and productivity! Ify is the collection of 4 must-needed productivity apps for your work!
  • Set your goals using Ify Tasks; write and draw your ideas down using Ify Notes and Ify Sketches; and create focusing sessions on your work using Ify Focus!
  • Even better, it's completely free and ad-free, with web version and browser extension version so you can get help from Ify anywhere, anytime!
  • Ify beta version officially launched on GitHub for testing!
New Feature: Tasks!
  • Tasks made easy. Add your works and goals to Ify Tasks and boost your productivity to finish them all!
  • You can now add tasks, complete them, and remove them from an organized tasks list!
  • New user interface: input bar at the bottom of the page, create task button in the input bar, new task list item style, and brand new elements placement.
New Page: Updates!
  • See what's new with Ify in the Updates page and try them out!
  • Keep track of Ify updates and know all the changes!
New User Interface and Functionalities
  • Brand new Ify interface with dark appearance.
  • Sidebar for easier navigation between Tasks, Notes, Sketches, and Focus.
  • New logo design and new links.
  • More comfortable section and link designs.
  • Bug fixes, including tasks not removing from localStorage, no ding sound effect when completed task, and scripting connections.
  • Overall performance improvements and minor updates.
